## API Introduction

The Akhshit Documentation API allows you to programmatically manage your documentation projects, including creating, updating, and retrieving documents.

## Authentication

All API requests require authentication using API keys.

<CodeGroup tabs="JavaScript,Python">
  ````javascript
const response = await fetch('/api/documents', {
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  }
});
  ````
  ````python
import requests

response = requests.get('/api/documents', 
  headers={'Authorization': f'Bearer {API_KEY}'})
  ````
</CodeGroup>

<Callout kind="alert">
  Keep your API key secure and never expose it in client-side code.
</Callout>

## Error Handling

<Expandable title="Common Error Codes" default-open="false">
  | Code | Description |
  |------|-------------|
  | 400 | Bad request - invalid parameters |
  | 401 | Unauthorized - invalid API key |
  | 404 | Not found - resource doesn't exist |
  | 500 | Internal server error |
</Expandable>
